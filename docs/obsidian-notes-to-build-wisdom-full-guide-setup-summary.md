# Obsidian Notes to Build Wisdom - Summary

This guide argues that most note-taking systems help you collect information, but they do not necessarily help you become wiser. The difference is that knowledge is collecting facts, while wisdom comes from connecting ideas across topics, experiences, values, and time.

Obsidian is presented as useful because it supports linking, backlinks, graph view, and repeated exposure to old ideas. The goal is not to build a perfect folder system or use fancy plugins. The goal is to create a thinking system where notes interact with each other and resurface over time.

## Core Philosophy

Most people write down useful ideas from books, videos, classes, or conversations, then store them away and forget them. This feels productive, but it often does not lead to deeper understanding.

The guide emphasizes:

- Real life does not fit into clean categories.
- Ideas from different areas should connect.
- Spaced repetition matters because people forget most new information quickly.
- Notes should be revisited, linked, and reorganized.
- The system should grow around your mind, not force your mind into someone else's structure.

## Basic Vault Setup

Create a new Obsidian vault. A vault is simply a folder where all notes are stored locally.

The recommended setup uses three main folders:

1. `1 - Raw Notes`
   - For unprocessed notes from books, podcasts, lectures, conversations, classes, and videos.
   - This acts as an inbox for learning.

2. `2 - Indices`
   - For major topic hubs.
   - Examples include entrepreneurship, relationships, greatness, meaning, philosophy, or self-discipline.
   - A topic becomes an index when many notes start linking to it.

3. `3 - Zettelkasten`
   - For processed thoughts, reflections, templates, tags, journal entries, mini essays, and personal insights.

The guide also suggests optionally creating a `Life Dashboard` note for tasks or planning.

## Templates

Templates save time and keep notes structured. Enable Obsidian's core Templates plugin, then create a templates folder inside the Zettelkasten folder.

Useful templates include:

### General Note

Use this for personal reflections or synthesized ideas.

Suggested fields:

- Date
- Tags
- Title
- References

### Academic Note

Use this for class or lecture notes.

Suggested fields:

- Date
- Course
- Tags
- Terms
- Notes
- Summary

### Raw Notes

Use this for unprocessed source material.

Suggested fields:

- Date
- Title
- References

Set a hotkey for inserting templates so new notes can be created quickly.

## Tags And Backlinks

The system uses Obsidian links as tags. For example:

```md
[[self-discipline]]
```

These tags are actual notes, which makes them more powerful than ordinary labels. When several notes link to `[[self-discipline]]`, the self-discipline note automatically shows backlinks to all related notes.

This means a note can belong to many topics at once without being trapped in one folder.

## Everyday Workflow

The recommended workflow is:

1. Capture raw notes from a source.
   - Example: a podcast, lecture, book, or video.
   - These notes should be mostly factual and unprocessed.

2. Review the raw notes.
   - Look for ideas that stand out.

3. Create a separate general note for each important insight.
   - This note should contain your own reflection, interpretation, and application.

4. Link the general note back to the original raw note under `References`.

5. Add relevant topic tags using Obsidian links.
   - Example: `[[Nietzsche]]`, `[[self-discipline]]`, `[[philosophy]]`.

6. Move processed general notes into the Zettelkasten folder.

Over time, the graph view and backlinks reveal connections between ideas that may not seem related at first.

## Creating An Index

When a tag accumulates many related notes, turn it into an index.

For example, if many notes link to `[[self-discipline]]`, open the self-discipline note and organize the linked notes into sections such as:

```md
## Self-discipline in philosophy

## Self-discipline in general ideas
```

Then move the note from the tags area into `2 - Indices`.

Indexes are flexible topic hubs. They combine automatic backlinks with manual organization, making them more useful than rigid folders.

## Main Takeaway

The power of Obsidian is not the software itself. It comes from using Obsidian to repeatedly revisit, connect, and refine ideas. A good system should help old notes resurface, reveal unexpected relationships, and turn scattered information into usable wisdom.

The guide's final advice is to experiment. There is no perfect universal Obsidian setup. The best system is one that grows around the way you think.
